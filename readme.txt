Contributors: Anderson.Makiyama

Tags: djs, musica, locutores

Requires at least: 3.0

Tested up to: 4.1.2

Stable tag: trunk



O Plugin Programação DJs permite você exibir no seu site o nome do DJ ou Locutor Atual, o Nome do programa e o Horário dele.


== Descrição ==

**Apresentação**

O Plugin permite você cadastrar todos os DJs e o nome dos programas que eles apresentam. Após isso você faz o cadastro da Programação escolhendo o dia da semana, o DJ e o horário de Atuação dele.

Um vez feita a programação, basta colocar o shortcode do plugin em posts ou páginas para que seja exibida a programação atual, o DJ e seu horário de atuação. 

Você pode cadastrar para ser exibido a foto do DJ ou Locutor, seu Facebook, seu Twitter, seu Instagram e seu Youtube. 

Além de poder colocar o shortcode em posts e páginas, você também pode adicionar o widget do plugin em qualquer sidebar disponível em seu Tema. 


**Sobre**

O Plugin Programação DJ foi escrito por Anderson Makiyama do [Plugin WP](http://plugin-wp.net)


== Instação ==

Para instalar e usar o plugin é muito simples:

1- Envie a pasta do plugin para dentro da pasta de plugins, que geralmente é /wp-content/plugins/

2- Faça login no painel de gerenciamento do teu blog WP, vá na seção plugin e ative o Plugin

3- Agora, no lado esquerdo do painel do blog serão exibidas as opções do plugin

4- Cadastre os DJs ou Locutores, Cadastre os Programas e Cadastre a Programação.

5- Para exibir no seu site insira o código de exibição que pode ser obtido na opção "Código Exibição" ou utilize o widget do plugin para exibir em algum sidebar de seu tema.
